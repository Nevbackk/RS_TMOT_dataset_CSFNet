## Track_code:
CPR_caronly为权重文件存放路径
train.py为训练文件
requirements.txt 指定本项目所需依赖
mot2coco.py为MOT转换为COCO格式的代码（若数据集默认COCO格式则无需运行）
path_config.py管理路径配置并设置训练参数
evaluate.py为测试及评估文件

## MOT_Dataset:
包含了图片数据，标注好了目标和观测平台的六维状态以及方位角和俯仰角。
